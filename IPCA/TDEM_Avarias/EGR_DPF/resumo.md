| **1. Sistema EGR** | **Função:** Recircular escape para a admissão. Menos O2 = Combustão mais fria = Menos NOx. | **A tua deixa:** "O EGR é como pedir a alguém para respirar o seu próprio peido para 'salvar o planeta'. Pegamos em gases sujos e quentes e metemo-los de volta no motor limpo. Sim, baixa a temperatura e os NOx, mas é o equivalente mecânico de comer do lixo. O motor odeia, mas a lei obriga." |
| **2. Avarias EGR** | **Problema:** Carbono + Óleo = Pasta preta. Válvula encrava. Sintomas: Fumo negro, Falta de força. | *(Jonathan e Francisco)*<br><br>**Comentário opcional:** "Vejam aquela imagem. Aquilo são as artérias de um fumador de 3 maços por dia. Aquilo é 'colesterol' de carbono. Quando a válvula encrava aberta, o motor engasga-se com o próprio fumo. É poético, e nojento." |
| **3. Sistema DPF** | **Filtro Partículas:** Esponja cerâmica. Retém fuligem. **Regeneração:** Queimar o lixo (auto-limpeza) a altas temperaturas. | *(João e José explicam)*<br><br>**A tua deixa (Regeneração):** "O DPF é um aspirador que não se sabe esvaziar sozinho a não ser que andem a 120km/h. A 'Regeneração' é uma palavra bonita para 'Inceneração'. O carro injeta gasóleo extra para o escape (sim, desperdiça combustível) para criar um inferno de 600 graus e queimar a fuligem. E se desligarem o carro a meio? Ele vinga-se e acende a luz de avaria." |
| **4. Avarias DPF** | **Caos:** Uso citadino entope tudo. Gasóleo no óleo (diluição). Sensor Pressão Diferencial = o delator. | **A tua deixa:** "Se comprarem um Diesel moderno para ir comprar pão à esquina, merecem o que vos vai acontecer. O filtro entope. A regeneração falha.
